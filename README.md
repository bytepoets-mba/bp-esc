# OpenRouter Balance Checker

**Internal Tool - BYTEPOETS GmbH Employees Only**

This is an internal tool for employees of BYTEPOETS GmbH to check their OpenRouter API balance.

## Features

- **No Login Required**: Simple macOS application with no authentication
- **OpenRouter API Key Storage**: Securely stores your OpenRouter API key in a local `.env` file
- **Balance Display**: Shows current OpenRouter balance (€ 123.00 for demo purposes)
- **Easy Exit**: Simple quit option to close the application

## Usage

1. Launch the application
2. Enter your OpenRouter API key when prompted
3. The key will be saved locally in a `.env` file
4. View your current balance
5. Use the quit option to close the application

## Security Note

Your API key is stored locally in a `.env` file. Ensure this file is not shared or committed to version control.
