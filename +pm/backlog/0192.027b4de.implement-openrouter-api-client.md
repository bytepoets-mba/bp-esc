# Title: Implement OpenRouter API Client

## Description

Fetch balance from OpenRouter API using saved API key.

## Priority
0300

## API Details

**Endpoint**: `GET https://openrouter.ai/api/v1/key`

**Headers**:
```
Authorization: Bearer {api_key}
Content-Type: application/json
```

**Response**:
```json
{
  "data": {
    // balance/usage info
  }
}
```

**Error**:
```json
{
  "error": "error message"
}
```

## Tasks

- [ ] Create Tauri command: `fetch_balance(api_key: String)`
  - Use reqwest crate for HTTP client
  - Set Authorization header
  - Parse JSON response
  - Return Result<BalanceData, String>

- [ ] Define BalanceData struct (serializable)
  - Extract relevant fields from API response
  - Format for display

- [ ] Handle errors:
  - Invalid API key (401)
  - Network errors
  - Parse errors
  - Timeout (10s limit)

## Acceptance Criteria

- [ ] Successfully fetches balance with valid key
- [ ] Returns error message for invalid key
- [ ] Handles network failures gracefully
- [ ] 10-second timeout implemented
- [ ] Response data accessible from frontend
